Form controls: Range

Scenario 7.3: Multi-touch Equalizer example

Back to Range control scenarios menu

Description

The Windows platform supports multi-touch to let you manipulate multiple controls at the same time. Normally this only works when the page isn't pannable or zoomable, but the range control wins over page panning and zooming, therefore in the equalizer example below you could move multiple range controls simultaneously.
