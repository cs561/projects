Description

There are three types of buttons: submit button, reset button, and standard button. The Submit button is activated when a user presses the Enter key while the focus is on another control in the same form that isn't a button, like a text field.

Clicking the reset button returns a field to its default state. In this example, it returns the field to its original text content.

The standard button can be set to call a JavaScript function. In this example, the button calls a simple arithmatic function.

Code

<!-- submit button
    <form onsubmit="sendMessage(); return false;">
        <input type="text" id="Text1" placeholder="Type your message here." />
        <button type="submit">Submit</button>
    </form>
        
        function sendMessage() {
            var data = document.getElementById("message").value;

            // Put your code here to send out message.
            // Using AJAX is recommended to avoid page switch.

            document.getElementById("sendMessageResult").innerText = 
                "Your message has been sent: " + data;
        }
    -->

<!-- reset button
    <form>
        <input type="text" value="Edit this content and then press reset." />
        <button type="reset">Reset</button>
    </form>
    -->

<!-- calculate button
    <button onclick="calculate1Plus1();">Calculate 1 + 1</button>

        function calculate1Plus1() {
            var result = 1 + 1;
            document.getElementById("calculate1Plus1Result").innerText =
                "The result is " + result + ".";
        }
    -->
