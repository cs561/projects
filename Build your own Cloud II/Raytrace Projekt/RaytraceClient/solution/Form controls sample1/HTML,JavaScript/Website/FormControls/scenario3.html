Scenario 3: Radio button

Back to scenarios menu

Description

Radio button can be used for single selection. All of the radio buttons under the same "name" attribute are part of the same group and only one of them can be selected at a time.

Code

<!-- radio button group 
    <label class="radioLabel horizontalLabelLayout">
        <input type="radio" name="sauce" checked />BBQ Sauce
    </label>
    <label class="radioLabel horizontalLabelLayout">
        <input type="radio" name="sauce" />Tomato Sauce
    </label>
    <label class="radioLabel horizontalLabelLayout">
        <input type="radio" name="sauce" />White Sauce
    </label>
    -->

<!-- change radio dot color
    input[type="radio"].differentRadioColor::-ms-check
    {
        color: green;
    }
    -->

<!-- thumb-up / thumb-down rating
    <input type="radio" name="thumbUpDownGroup" class="thumbUpDown thumbUp" />
    <input type="radio" name="thumbUpDownGroup" class="thumbUpDown thumbDown" />

    /* reset default styles */
    input[type="radio"].thumbUpDown,
    input[type="radio"].thumbUpDown::-ms-check,
    input[type="radio"].thumbUpDown:hover::-ms-check,
    input[type="radio"].thumbUpDown:active::-ms-check,
    input[type="radio"].thumbUpDown:hover:active::-ms-check,
    input[type="radio"].thumbUpDown:hover:active:disabled::-ms-check
    {
        color: transparent;
        border: 0px;
        background-color: transparent;
        padding: 0px;
        margin: 0px;
    }

    input[type="radio"].thumbUpDown
    {
        width: 60px;
        height: 60px;
        border: 0px;
        border-radius: 0px;
        margin: 0px;
        background-repeat: no-repeat;
        background-position: center center;
        background-size: 100% 100%;
    }

    input[type="radio"].thumbUpDown:hover
    {
        opacity: 0.8;
    }

    input[type="radio"].thumbUp
    {
        background-image: url("../images/thumb.svg");
    }


    input[type="radio"].thumbUp:checked
    {
        background-image: url("../images/thumb_filled.svg");
    }


    input[type="radio"].thumbDown
    {
        background-image: url("../images/thumbDown.svg");
    }


    input[type="radio"].thumbDown:checked
    {
        background-image: url("../images/thumbDown_filled.svg");
    }
    -->
