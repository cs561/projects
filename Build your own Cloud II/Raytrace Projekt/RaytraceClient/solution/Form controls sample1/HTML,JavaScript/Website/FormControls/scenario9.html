Form controls

Scenario 9: Form layout

Back to scenarios menu

Description

This example shows a recommended way to layout form control margins and reading order for a two column form. The reading and keyboard tab order goes from user name to password, then to email, and so forth. If there aren't too many controls, you can do a one column form layout instead.
