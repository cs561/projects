Description

This sample provides examples of input and form controls, including the HTML5 Progress element and the input type=”range” control. CSS styling lets you customize controls to match the look and feel of your app. Using CSS, custom background images or SVG graphics can be applied to "skin" your controls to go with the design of your page.

The checkbox button, and radio button examples show how elements can now be styled with standard CSS styling to let you match the visual design of your app.

The Text input control example shows support more input types, including number, telephone, email, and URL. The select control example demonstrates using and styling single and multi-line select boxes and drop-down lists.

The File upload control example show how it simplifies supporting single and multi-file selection. User actions like selecting pictures to upload to a photo site can be easier using the accept attribute to specify acceptable file mime types. Clicking the "browse" button brings up the file dialog so you can select files from your local disk. Files can be sent to the server as part of the formdata when the form is submitted.
 
The Range example shows how to use the input range control to create a slider to pick non-precise numbers quickly. The input range can be customized for your app’s needs with the min, max, and step attributes. The example shows how to use and style the slider control for custom looks. 

The Progress example shows how to show activity or a task’s progress, for example, when loading a large file, or doing intensive calculations. The progress control can be either indeterminate (just showing activity) when no value attribute is provide, or a visual progress toward a goal when a value is provided. The progress control defaults to a floating point range of 0 to 1, but you can set the max attribute to any positive number. The example shows how to make a circular or linear progress bar.

The Form layout example shows how to use CSS grid and columns to lay out fields on a webpage.
